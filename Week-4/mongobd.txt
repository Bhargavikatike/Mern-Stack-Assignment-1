*update*
1. upate one: db.emps.updateOne({empno:100},{$set:{name:"ravi kumar"}})
2. add new property: db.emps.updateOne({empno:100},{$set:{"address.street":"uppal"}})
3. update : db.emps.updateOne({empno:100},{$set:{"address.city":"vizag"}})
4. delete field: 
string: db.emps.updateOne({empno:100},{$unset:{name:""}})
numb:db.emps.updateOne({empno:100},{$unset:{"address.pincode":""}})
5.add : db.emps.updateOne({empno:100},{$set:{firstname:"katike"}})


array update oprns
$addToSet :add ele's to an array if they do not already exist
db.emps.updateOne({empno:100},{$addToSet:{skills:"angular"}})
$push add :an ele to array
db.emps.updateOne({empno:100},{$push:{skills:"angular"}})

insert multiple ele's into array
$push--insert $each--pick each multiple ele and push into array  
db.emps.updateOne({empno:100},{$push:{skills:{$each:["node.js","java"]}}})


if u know frst or last then pop
pop: last element remove : db.emps.updateOne({empno:100},{$pop:{skills:1}})
1st ele db.emps.updateOne({empno:100},{$pop:{skills:-1}})
if u doint know 1st and last then pull:
pull: db.emps.updateOne({empno:100},{$pull:{skills:"css"}})
